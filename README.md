# 📊 Indicadores das Caixas de Contribuição

Aplicativo web para escalonamento voluntário dos indicadores de caixas de contribuição.

## 🚀 Funcionalidades

- 📝 Preenchimento por turnos e vagas
- 💾 Salvamento automático no Firebase
- 🔄 Sincronização em tempo real entre múltiplos usuários
- 📱 Design responsivo
- 🔒 Sem necessidade de login

## 🛠️ Tecnologias

- HTML/CSS/JavaScript
- Firebase Realtime Database
- GitHub Pages

## 📦 Como usar

1. Acesse a URL do projeto
2. Selecione um dia (Sexta, Sábado ou Domingo)
3. Preencha os campos disponíveis
4. Os dados são salvos automaticamente

## 🔧 Configuração

1. Copie `firebase-config.example.js` para `firebase-config.js`
2. Adicione suas credenciais do Firebase
3. Publique no GitHub Pages
